# Jetter

Jetter is an iOS app for flight lookup and jet-lag planning.

## Local setup

1. Copy `Jetter/Configuration/APIKeys.plist.example` to `Jetter/Configuration/APIKeys.plist`.
2. Replace `YOUR_RAPIDAPI_KEY` with your AeroDataBox RapidAPI key.
3. Open `Jetter.xcodeproj` in Xcode and run the app.

`Jetter/Configuration/APIKeys.plist` and `Jetter/Configuration/Secrets.xcconfig` are ignored so local secrets do not get committed.
